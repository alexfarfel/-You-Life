# FarfLife - Habit Tracker

A gamified habit tracking Progressive Web App (PWA) with XP system, streaks, and customizable tasks.

## Features

- Daily XP progress bar with customizable goals
- Streak tracking with fire emoji
- Two task types: Daily Essentials (once per day) and Quests (repeatable)
- Weekly progress visualization
- Full customization (add/edit/delete tasks)
- Satisfying animations and confetti celebrations
- Weekly Review with charts and statistics
- PWA support - installable on mobile devices
- Offline support
- Data persists in localStorage

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- No frameworks or build tools required
- Progressive Web App (PWA) with Service Worker

## Local Development

Simply open `index.html` in your browser, or serve with any static file server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```
